# 强安兴企安全园地生文助手

> v4.x — 选题日历 + 栏目模板 + AI 自由生成兜底

## 一句话介绍

输入一个安全话题，自动联网搜索权威站点的最新资料，调用大模型生成带标题、正文、配图方案和审核提醒的完整图文草稿。内置选题日历和 5 种栏目模板，文员打开就知道该写什么、怎么写。

## 核心能力

| 维度 | 说明 |
|------|------|
| **选题日历** | 12 个月安全宣传主题预设（汛期三防 / 安全生产月 / 冬季三防等），自动推荐当月选题，搜索关键词自动匹配当月热点 |
| **栏目模板** | 5 种固定栏目一键切换：事故案例分析 / 班前会要点 / 安全知识问答 / 本周安全之星 / 季节性提醒，每种都有专门的结构化提示词 |
| **搜索源** | 7 站：应急管理部、矿山安全监察局、煤炭报、煤矿安全网、安全文化网、煤炭资讯网、国家卫健委 |
| **搜索回退** | 安全站点无结果时自动回退通用 Bing 搜索，确保自定义主题也能搜到内容 |
| **模型支持** | DeepSeek 直连 + OpenRouter 14 品牌 + 智谱 GLM，共 80+ 模型可选 |
| **生成质量** | 800-1500 字结构化正文，含分节标题、真实案例引用、操作步骤指引、审核提醒 |
| **AI 兜底** | 搜索无结果时改由 AI 基于主题自由生成，不再出重复模板文章 |
| **云端自动抓取** | GitHub Actions 每日定时抓取 7 大安全新闻源最新文章，存入 GitHub Gist 素材库 |
| **离线兜底** | 素材 localStorage 缓存 30 天，没网也能看素材列表 |
| **零构建静态工具集** | 文件夹整体复制使用，无需构建，即开即用 |

## 优点

- **零安装**：纯 HTML/CSS/JS 静态文件集，浏览器直接打开，不依赖 Node/构建工具/数据库
- **真实联网搜索**：不走虚构，搜索-抓取-生成全链路真实执行，结果标注来源链接
- **多模型自由切换**：API 设置面板统一管理，品牌-模型-密钥一键切换
- **不造假**：API 调用失败时降级为本地兜底生成，不编造内容
- **每日自动抓取**：GitHub Actions 定时管道每日抓取 7 大安全新闻源最新文章，自动去重分类存入 Gist。管道失败自动建 Issue 告警，不做静默失败
- **结构化输出**：JSON 格式约束模型输出，正文/配图/审核三件套标准化

## 缺点与限制

- **依赖 CORS 代理**：浏览器端搜索必须经过 CORS 代理转发，代理挂了搜索就不可用（当前使用腾讯云 SCF）
- **Bing 搜索不稳定**：HTML 解析依赖 Bing 页面 DOM 结构，Bing 改版需更新正则
- **搜满 15 秒无结果**：每个搜索源最长等 15 秒，7 站并发搜索（Promise.all），全超时约 15 秒通知用户"没搜到"
- **全文抓取噪音大**：任意网页抓取 HTML 后用正则提取正文，导航/页脚/广告混入不可避免，靠 6000 字截断缓解
- **模型 API 不可控**：第三方 API 超时/限流/格式异常是常态，已做 60 秒超时 + 异常吞没 + 格式兼容解析三重防护
- **生成质量依赖搜索结果**：搜不到好内容 → 模型只能"硬编"或输出空泛内容
- **CORS 预检敏感**：浏览器 fetch 中任何自定义 HTTP 头会触发 OPTIONS 预检，代理必须声明 `Access-Control-Allow-Headers`
- **无用户系统**：API Key 存在浏览器 localStorage，换设备需重新填写
- **模型 API 路径不统一**：DeepSeek 用 `/v1/chat/completions`，GLM 用 `/api/paas/v4/chat/completions`，新模型需要手动适配 `chatEndpoint`

## 技术栈

- 前端：纯 HTML/CSS/JS（原生，零框架）
- 搜索：Bing cn.bing.com（HTML 解析 + 正则提取）
- CORS 代理：Express（Node.js），部署在腾讯云 SCF / Railway
- 素材来源：GitHub Gist（raw JSON，云端定时抓取管道）
- 模型 API：OpenAI 兼容接口（`/v1/chat/completions`），直连 DeepSeek + OpenRouter 中转

## 快速开始

1. 用浏览器打开 `强安兴企安全园地生文助手.html`
2. 点击右上角齿轮 → API 设置
3. 选择模型品牌和型号，填入 API Key
4. 点击"获取安全素材"加载云端素材
5. 输入话题或点选素材 → 点击"自动生成文章"
