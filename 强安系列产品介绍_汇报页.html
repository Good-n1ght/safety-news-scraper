Product 01

强安兴企安全园地 AI 生文助手

输入安全话题，自动联网搜索 7 个权威站点最新资料，调用大模型生成带标题、正文、配图方案和审核提醒的完整图文草稿。

  • 实时联网搜索不再依赖预置素材——每次按下生成按钮，实时去应急管理部、矿山安监局、煤炭报、煤矿安全网、安全文化网、煤炭资讯网、国家卫健委 7 站抓取当日最新文章。结果标注来源链接，可溯源可核实
  • AI 自动生成完整稿件基于搜索到的真实资料，生成 800-1500 字结构化正文（4-5 个小节 + 操作指引），含标题、封面描述、配图关键词、审核提醒，不是套模板，每次内容都不同
  • 每日自动抓取GitHub Actions 定时管道每日自动抓取 7 大安全新闻源最新文章，去重分类后存入 GitHub Gist 云端素材库。前端拉取 + 本地 30 天缓存，离线也能浏览。管道失败自动创建 Issue 告警，不做静默失败
  • 15+ 模型品牌支持内置 DeepSeek + OpenRouter + 智谱 GLM 三大通道，80+ 模型可选。API 设置面板统一管理品牌-模型-密钥，一键切换
  • 关键词全覆盖搜索关键词覆盖安全事故/班组建设/现场实训/职工健康/职业病/心脑血管/高血压/急救/饮食/戒烟限酒/季节防病/节日安全
  • 扩展系统内置钩子系统（10 个钩子点），覆盖搜索/抓取/生成/渲染全链路。新增搜索源或自定义逻辑无需改核心代码，复制扩展模板修改即可
  • 导出分发一键复制 / 导出 PNG 卡片，微信群转发、打印张贴

当前局限

  • 依赖 CORS 代理转发搜索请求，已配置腾讯云 SCF + allorigins.win 双代理自动 fallback
  • Bing HTML 解析依赖页面 DOM 结构，搜索引擎改版可能导致搜索结果提取失败
  • 生成质量受搜索结果影响较大——搜不到好内容则文章质量下降
  • API Key 存浏览器 localStorage，换设备需重新填写
  • 7 站采用 Promise.all 并发搜索，最慢约 15 秒，全站超时体验不佳
